## Overview

Helios is a custom-designed LED controller based on the ESP32 platform. Supporting WLED firmware, it provides a complete solution for controlling LED strips with additional features like temperature monitoring and fan control for safe long-term operation.

## Features

- ESP32-based design
- WLED firmware compatibility
- Temperature sensor integration
- Fan control for cooling
- Multiple hardware revisions
- Custom PCB design with KiCad
- WiFi connectivity
- Mounting holes for installation

## Technologies

- ESP32 microcontroller
- KiCad for PCB design
- WLED firmware
- LED strip control
- Temperature monitoring
- WiFi integration
